## Tạo website blog free bằng github

- Set name the same as username
- Git clone repo 
- Git clone repo này: https://github.com/vietnamcrypto/vietnamcrypto.github.io
- Copy paste bỏ qua

- Git commit -m "up"
- Git push origin main
- Done
